GitHub Pages + GitHub Actions

Bulk email flow app hosted on GitHub, with Google Sheets as the only source of truth.

This version keeps your UI on GitHub Pages and runs import, send, and follow-up jobs through GitHub Actions using repository secrets.

Open Actions Secrets Guide

1. Initialize sheet

Run MailFlow Runner with task init-sheet.

2. Import contacts

Commit your CSV/XLSX into imports/, then run task import-file.

3. Send initial emails

Choose your campaign and template JSON, then run send-initial.

4. Send follow-ups

Run send-followup with the required stage from 1 to 10.

Required secrets

  • GOOGLE_CREDENTIALS_JSON
  • MASTER_SHEET_ID
  • MASTER_SHEET_TAB
  • SMTP_HOST
  • SMTP_PORT
  • SMTP_SECURE
  • SMTP_USER
  • SMTP_PASS
  • SMTP_FROM_EMAIL
  • SMTP_FROM_NAME

Template placeholders

{{name}}

{{customLine}}
